# CryptoTracker - Клон CoinMarketCap

Современный веб-трекер криптовалют, построенный на Vue.js 3 с использованием Composition API и бесплатного API CoinGecko.

## Возможности

- ✅ Отображение топ-50 криптовалют по рыночной капитализации
- ✅ Актуальные данные о ценах в реальном времени
- ✅ Изменения цен за 24 часа и 7 дней
- ✅ Рыночная капитализация и объем торгов
- ✅ Темная тема в стиле CoinMarketCap
- ✅ Адаптивный дизайн

## Технологии

- **Vue.js 3** - прогрессивный JavaScript фреймворк
- **Vue Router** - маршрутизация
- **Axios** - HTTP клиент для запросов к API
- **Vite** - быстрая сборка и dev-сервер
- **CoinGecko API** - бесплатный API для данных о криптовалютах

## Установка и запуск

### Требования

- Node.js >= 20.18.2
- npm >= 10.8.2

### Установка зависимостей

```bash
npm install
```

### Настройка переменных окружения

Создайте файл `.env` в корне проекта (можете скопировать `.env.example`):

```bash
cp .env.example .env
```

Файл `.env` содержит:
```env
VITE_API_BASE_URL=https://api.coingecko.com/api/v3
```

### Запуск dev-сервера

```bash
npm run dev
```

### Сборка для production

```bash
npm run build
```

### Предпросмотр production сборки

```bash
npm run preview
```

## Принципы разработки

- **Компонентный подход** - каждый UI элемент - это отдельный переиспользуемый компонент
- **DRY (Don't Repeat Yourself)** - избегаем дублирования кода
- **KISS (Keep It Simple, Stupid)** - простота и читаемость кода
- **Composition API** - современный подход к написанию Vue компонентов

## API

Проект использует бесплатный [CoinGecko API](https://www.coingecko.com/en/api) для получения данных о криптовалютах.

Лимиты бесплатного API:
- 10-30 запросов в минуту
- Нет необходимости в API ключе
